Antti's vim configuration files.

Installation (Unix/OS X):

>     $ cd
>     $ git clone git://codaset.com/antti/vimfiles.git vimfiles
>     $ cd vimfiles
>     $ ./install.sh

Installation (Windows):

>     > cd %USERPROFILE%
>     > git clone git://codaset.com/antti/vimfiles.git vimfiles
>     > cd vimfiles
>     > install
