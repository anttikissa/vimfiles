Antti's vim configuration files.

Installation (Unix/OS X):

    $ cd
    $ git clone git://github.com/anttisykari/vimfiles.git vimfiles
    $ cd vimfiles
    $ ./install.sh

Installation (Windows):

    > cd %USERPROFILE%
    > git clone git://github.com/anttisykari/vimfiles.git vimfiles
    > cd vimfiles
    > install
